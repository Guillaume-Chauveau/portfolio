Application de gestion en Java

Technologies : Java, Swing, JDBC

Contexte & Objectifs

Développement d’une application Java permettant la gestion de données (utilisateur, groupe, proposition). Le projet était orienté objet et incluait une base de données connectée via JDBC.

Fonctionnalités

  • Interface utilisateur en Java Swing.
  • Connexion à une base MySQL avec JDBC.
  • CRUD complet (ajout, lecture, mise à jour, suppression).
  • Recherche et filtrage avancés.

Résultats

Projet complet qui m’a permis de renforcer ma maîtrise de Java, de la POO et de la connexion à une base de données.
