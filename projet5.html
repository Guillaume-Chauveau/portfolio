Application de rétrocataloguage de documents

Technologies : Python, PySide6, Qt Designer, fichiers texte, export MARC

Contexte & Objectifs

Ce projet GitHub correspond à une application de rétrocataloguage de documents scientifiques. L’objectif est de faciliter l’édition et le traitement de fiches bibliographiques extraites de documents scannés, avec une interface graphique dédiée au catalogage semi-automatisé.

Fonctionnalités principales

  • Interface graphique Python avec PySide6 et Qt Designer.
  • Gestion de fiches bibliographiques et de caractéristiques multiples.
  • Traitement de sorties issues de modèles LLM pour préremplir les formulaires.
  • Export au format MARC simplifié.
  • Navigation entre catalogue, formulaire, rendu et paramètres.
  • Gestion du zoom, sauvegarde, validation et affichage de progression.

Points forts du projet

Le projet met en avant une architecture modulaire, la séparation interface/logique, ainsi qu’une logique de maintenance et d’extension facilitée pour de nouveaux types de documents ou formats d’export.
