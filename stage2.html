Application de rétrocataloguage de documents

Projet : Application Python PySide6 développée sur GitHub.

Contexte : Développement d’une interface graphique pour faciliter le rétrocataloguage de documents scientifiques à partir de sorties LLM et d’images scannées.

Objectif

Créer une application modulaire permettant d’éditer, valider et exporter des fiches bibliographiques de manière structurée.

Missions réalisées

  • Conception d’une interface utilisateur avec PySide6 et Qt Designer.
  • Création de classes Python pour la gestion des fiches et des caractéristiques.
  • Mise en place d’un flux de traitement et d’export au format MARC simplifié.
  • Préparation d’un manuel technique pour documenter l’architecture et l’utilisation.
  • Organisation du projet sur GitHub avec un dépôt public.

Résultat

Une application fonctionnelle de rétrocataloguage, avec une structure modulaire, un guide d’utilisation et un dépôt GitHub prêt à être consulté.
